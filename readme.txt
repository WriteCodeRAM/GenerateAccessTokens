 step 1: install the necessary dependencies by running 'npm install' in the integrated terminal
 step 2: adjust the base variable (generateToken.js) to be your endpoint for generating a token
 step 3: ensure .env file contains valid credentials (CLIENT_ID / CLIENT_SECURITY)
 step 4: start server (app) by running 'npm start' in the integrated terminal
 step 5: hit one of the provided routes in your browser or using a client like postman
 step 6: observe the terminal
